# Professional Network Documentation

## Overview
This dimension is dedicated to professional networking, endorsements, and building industry connections.

## Architecture
- **Path**: `/src/dimensions/professional-network/`
- **Components**: Isolated to prevent cross-contamination with other dimensions.
- **State**: Manages connections, profile views, and messages.

## Limitations & Roles
- **Student**: Can connect with peers and request endorsements.
- **Teacher**: Can endorse students and build expert profiles.
- **Personal**: Can build a professional portfolio and network independently.
- **Admin**: Full platform moderation and network analytics access.
