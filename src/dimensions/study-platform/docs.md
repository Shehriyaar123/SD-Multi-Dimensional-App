# Global Study Platform Documentation

## Overview
This dimension is dedicated to borderless learning, connecting mentors and peers globally.

## Architecture
- **Path**: `/src/dimensions/study-platform/`
- **Components**: Isolated to prevent cross-contamination with other dimensions.
- **State**: Manages its own learning paths and peer connections.

## Limitations & Roles
- **Student**: Can access modules and request mentorship.
- **Teacher**: Can create modules and accept requests.
- **Personal**: Independent learner with access to public modules.
- **Admin**: Full platform moderation and analytics access.
