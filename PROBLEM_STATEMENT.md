# Problem Statement: Lumina AI (Penta Ecosystem) - A Unified Multi-Dimensional Intelligence Platform

## 1. Executive Summary
In the modern digital landscape, students, educators, and professionals are forced to navigate a fragmented ecosystem of tools for learning, coding, career development, and content creation. This fragmentation leads to cognitive overload, data silos, and inefficient workflows. **Lumina AI (The Penta Ecosystem)** is designed as a centralized, self-contained intelligent platform that unifies these disparate workflows into a single, NLP-driven tool orchestration system. By integrating six core dimensions and a future-ready recruitment layer, Lumina AI provides a seamless transition between academic mastery and professional excellence.

## 2. The Six Dimensions of Intelligence

### A. Dimension 1: Poly Argu-Minds (Global Study Platform)
*   **Core Objective:** To foster critical thinking and borderless collaboration through structured debates and AI-assisted learning.
*   **Key Features:**
    *   **AI Tutor (Voice-Enabled):** A real-time voice-based AI system utilizing browser audio APIs for Speech-to-Text (STT) and Text-to-Speech (TTS) interactions.
    *   **Study Groups:** Collaborative chat environments with embedded AI tools for real-time translation, summarization, and explanation.
    *   **Neural Peer Matching:** Algorithms that connect mentors and peers globally based on cognitive compatibility and skill gaps.

### B. Dimension 2: CodeForge (Elite Coding Arena)
*   **Core Objective:** To provide a high-stakes, sandboxed environment for competitive programming and industry-standard project simulations.
*   **Key Features:**
    *   **Integrated Editor:** A Monaco-based coding workspace with real-time execution across 24+ languages.
    *   **AI Code Assistant:** Intent-driven tools for `GENERATE_CODE`, `DEBUG_CODE`, `OPTIMIZE_CODE`, and `EXPLAIN_CODE`.
    *   **Execution Console:** Secure, Docker-based runtimes providing instant feedback and AI-driven error analysis.

### C. Dimension 3: Resume Architect (Professional Network)
*   **Core Objective:** To transform professional narratives into verifiable, ATS-optimized portfolios using advanced NLP.
*   **Key Features:**
    *   **Semantic Extraction:** Automatically identifies hidden skills from user activity across the ecosystem.
    *   **ATS Optimization:** Real-time scoring against global job descriptions to maximize interview success rates.
    *   **Verified Endorsements:** A trust-based network for skill validation and industry connections.

### D. Dimension 4: Verified Job Network (Career Launchpad)
*   **Core Objective:** To eliminate recruitment friction through a trustless, manually verified hiring pipeline.
*   **Key Features:**
    *   **KYC Verification:** Every company and job post is vetted by the Admin layer to prevent scams and ghost jobs.
    *   **Direct-to-CTO Messaging:** Secure communication channels between top talent and technical leadership.
    *   **Blockchain Credentials:** Verifiable candidate history and salary benchmarks.

### E. Dimension 5: Digital Knowledge Library
*   **Core Objective:** To index the world's collective intelligence using semantic search and intent-based retrieval.
*   **Key Features:**
    *   **Vector Indexing:** Sub-second semantic retrieval of research papers, tutorials, and documentation.
    *   **3D Visualization:** Interactive WebGL-powered modules for complex scientific and engineering concepts.
    *   **LLM Synthesis:** Automated synthesis of answers from multiple verified sources with inline citations.

### F. Dimension 6: Creative Studio
*   **Core Objective:** To provide a multi-tool content generation and transformation ecosystem for creators.
*   **Key Features:**
    *   **Content Generation:** Structured output for blogs, social media, and professional emails.
    *   **Transformation Pipeline:** Sequential AI actions (e.g., Summarize -> Rewrite -> Change Tone -> Translate).
    *   **Advanced Utility:** Standalone TTS, Idea Generation, and Tone Adjustment engines.

## 3. Future Roadmap: Recruiter Examination Dimension
*   **Objective:** To provide recruiters with a dedicated layer for technical assessments and automated candidate evaluation.
*   **Implementation:** Proctored coding tests, AI-graded subjective answers, and real-time candidate performance analytics integrated directly into the Job Network.

## 4. Governance and Role-Based Access Control (RBAC)

### A. Role Hierarchy
*   **Admin:** Full system control, user management, and content moderation. Responsible for uploading DSA problems and verifying recruiters.
*   **Professional:** Full access to all learning and creative dimensions (Study, CodeForge, Creative Studio, Job Portal).
*   **Teacher:** Instructional access to create content, manage study groups, and guide student progress.
*   **Student:** Guided learning access with strict communication boundaries.

### B. Communication Restrictions
*   **Student Protection:** **Student-to-student direct chat is strictly prohibited** to ensure a focused learning environment.
*   **Moderation:** A centralized NLP pipeline performs real-time abuse detection and inappropriate language filtering across all interactions.

## 5. Technical Architecture
*   **Frontend:** React (TypeScript) with Framer Motion for immersive, motion-driven UI/UX.
*   **Backend:** Spring Boot (Java) for robust orchestration and secure business logic.
*   **AI Infrastructure:** A custom Python-based NLP microservice for intent classification and entity extraction, eliminating dependency on external AI APIs.
*   **Persistence:** PostgreSQL for structured data, logs, and interaction history.
*   **Real-time:** WebSockets for live chat, execution feedback, and notifications.
